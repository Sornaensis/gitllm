## [Unreleased]

### Added
- Initial project scaffold with Stack (lts-24.2)
- MCP protocol implementation (JSON-RPC 2.0 over stdio)
- Comprehensive git tool definitions covering 60+ operations
- Tool categories: status, log, diff, branch, commit, staging, remote,
  stash, tag, merge, rebase, cherry-pick, worktree, submodule, config,
  blame, bisect, clean, reset, reflog, search, patch, archive, hooks, inspect
- Haskell-based install script for Windows and Linux
- MCP configuration templates for Claude Code and GitHub Copilot
- Design plan with phased implementation roadmap
